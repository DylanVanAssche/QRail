Detailed Description

An RouterEngine::NullJourney contains a empty journey.

A Journey holds all the necessary information about a journey for the connection between 2 stops. A journey can keep track of multiple routes to get from A to B.

Warning
Receiving a NullJourney from the QRail library indicates that something must be terrible wrong! Check you debug output carefully.
