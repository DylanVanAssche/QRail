Detailed Description

An LiveboardEngine::Factory allows you to generate LiveboardEngine::Board objects.

The factory design pattern allows you to create Board objects in an easy way. Several modes are available to fetch your Board.

Retrieves a liveboard by a station URI.

Parameters
constQUrl &url
constQRail::LiveboardEngine::Board::Mode &mode This is an overloaded member function, provided for convenience. It differs from the above function only in what argument(s) it accepts.

Retrieves a QRail::LiveboardEngine::Board for a station given by it's URI. The QRail::LiveboardEngine::Board::Mode &mode parameter determines if the the liveboard should contain all the arrivals, departures of the station. Calling this method will retrieve a QRail::LiveboardEngine::Board for the current time until the current time + 0.5 hour.
